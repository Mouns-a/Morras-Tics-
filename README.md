# Morras Tics

Morras Tics es un club que ayuda a que más mujeres se unan a la tecnología. Nos capacitamos en ciberseguridad y expandimos nuestro conocimiento a otros lugares.

## Portada web (estilo cyberpunk)

La portada se encuentra en `index.html` (replicada también en `morras`) e incluye:

- Fondo animado con gradientes dinámicos.
- Efecto visual de nodos y conexiones en canvas.
- Título y subtítulo con efecto glow.
- Botón principal de llamada a la acción.

### Cómo verla localmente

```bash
python3 -m http.server 4173
```

Luego abre: `http://127.0.0.1:4173/index.html`.
